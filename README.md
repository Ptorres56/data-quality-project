# Data Cleaning & Quality Analysis Project

## Overview
This project demonstrates how to clean and validate raw data using Python and pandas. The goal is to improve data accuracy, consistency, and reliability.

## Features
- Removed duplicate records
- Handled missing values using averages
- Standardized data formats
- Validated email fields
- Generated a data quality score
- Produced a cleaned dataset for analysis

## Technologies Used
- Python
- pandas

## Output
- Cleaned dataset (cleaned_data.csv)
- Data quality report printed to the console

## Purpose
This project demonstrates practical data cleaning, validation, and reporting techniques relevant to data quality and data analyst roles.
